Donations

The Donations page lists the gifts that have come in through Ergatas, with tools to sort, filter, and export them. Organization administrators see donations across the profiles they manage; an individual worker sees the donations to their own profile.

Reading the list

Each row shows the donor's name (for administrators), the date, and the amount. A cycle icon next to the amount marks a recurring donation, and a status label shows the payment state. Click the expand arrow on a row to see more detail:

  • Donor info — name and contact details, when provided.
  • Type — one-time or recurring.
  • Whether the donation was made on site (through Ergatas) or off site.
  • The payment status.

Sorting and filtering

  • Use the sort dropdown to order by date (sooner or later) or by amount (big or small).
  • Click Filter by Date to set a Start Date and/or End Date and show only donations in that range. Clear a date with the × button to remove that limit.
  • Long lists are paginated. Use the page controls to move between pages, and the Page size dropdown to show 10, 50, or 100 per page.

Exporting

Use the download button group to export your donations for bookkeeping:

  • CSV — a spreadsheet-friendly file of all the listed donations.
  • QIF — a format for accounting software. The dropdown offers a QIF export per donor name when there's more than one.

Exports respect the current date filter, so set the range first if you only want part of your history.
